# README

Welcome to the Gossip Project

git clone repo,
cd repo,
bundle install,
rails db:create,
rails db:migrate,
rails db:reset,
rails server,

go to localhost:3000

Versions required : 
Ruby 2.5.1
Rails 5.2.4, >= 5.2.4.1
